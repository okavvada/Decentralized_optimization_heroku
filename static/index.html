This tool visualizes a published modeling algorithm for identifying the optimal scale for decentralized non-potable water reuse. The user can set his/hers desired treatment performance by defining the treatment performace curve above, selecting the appropriate electricity mix and any potential direct GHG emissions from treatment.

By selecting the appropriate metric (energy, GHG or cost) for the assessment and clicking on the desired location, the algorithm estimates the optimal system size for water reuse implementation at the specific location, which minimizes the selected metric. The buildings that contribute to the optimal system size are identified as "connected".
